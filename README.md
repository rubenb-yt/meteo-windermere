# 🌦️ Windermere Meteo Live

Dashboard meteorologica multi-modello per **Windermere, LA23** (Lake District).

Aggrega in tempo reale **7 modelli NWP** (Met Office UKV, ECMWF, GFS, ICON, GEM, Météo-France, JMA) tramite Open-Meteo API, con:

- Grafici continui con bande d'incertezza inter-modello
- Sinossi intelligente con rilevamento passaggi frontali
- Semaforo giornaliero e consigli pratici
- Timeline oraria interattiva con tooltip dettagliati
- Funziona offline (PWA con Service Worker + IndexedDB)

## 📱 Installazione su iPhone

1. Apri `https://rubenb-yt.github.io/meteo-windermere/` su Safari
2. Tocca **Condividi** (↑) → **Aggiungi alla schermata Home**
3. L'app appare come icona nativa
